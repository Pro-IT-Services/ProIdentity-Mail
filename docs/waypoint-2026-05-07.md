## Backup And Restore

Commands available:

```bash
mailctl backup --output-dir /var/backups/proidentity-mail
mailctl backup --output-dir /var/backups/proidentity-mail --prune-after
mailctl backup-prune --dir /var/backups/proidentity-mail
mailctl backup-verify --archive /var/backups/proidentity-mail/<archive>.tar.gz
mailctl restore --archive /var/backups/proidentity-mail/<archive>.tar.gz
mailctl restore --archive /var/backups/proidentity-mail/<archive>.tar.gz --apply --target-root /tmp/restore-stage --overwrite
mailctl restore --archive /var/backups/proidentity-mail/<archive>.tar.gz --apply --live --overwrite
```

Retention defaults:

- 7 daily
- 4 weekly
- 12 monthly
